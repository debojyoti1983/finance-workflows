# Finance Workflows — Claude Code Marketplace

A Claude Code plugin marketplace with skills for the Personal Finance Tracker app.

## Install

**Step 1 — Add this marketplace:**
```
/plugin marketplace add github:debojyoti1983/finance-workflows
```

**Step 2 — Install the plugin:**
```
/plugin install finance-workflows
```

## Plugins

### finance-workflows
Skills for managing the Personal Finance Tracker app:
- **CardFlow** — Guides you through adding/managing credit card transactions
- **Setup** — Step-by-step installation and setup guide for the app
